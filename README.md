# To-Do List Website

A clean and aesthetic to-do list web application designed to help users manage tasks one step at a time.

## Features
- Add new tasks easily
- Delete tasks with a single click
- Mark tasks as completed
- Dynamic progress bar showing task completion percentage
- Motivational quote for better user experience
- Minimal and visually pleasing UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
- Users can add tasks using the input field
- Tasks can be removed individually
- The progress bar updates automatically based on completed tasks
- Tasks remain saved even after page refresh using localStorage
